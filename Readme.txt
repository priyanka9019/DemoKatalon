	Tools used for this task
	1. Microsoft Visual Studio Professional 2022 (64-bit)
	2. Google chrome browser Version 110.0.5481.178 
	
	Packages used 
	1. Selenium webdriver(4.9.1)
	2. Nunit (3.13.2)
	3. Specflow Nunit(3.9.40)
	3. Extent reports(4.1.0)
	4. Chromedriver(113.0.5672.6300)

	Steps to run from Visual studio 2022
	1. Open the DemoKatalon.sln in visual studio 2022
	2. Build the solution
	3. Go to menu View->Test Explorer
	4. Expand tests and right click on VerifyCartFunctionalityWithFourItems
	5. Click Run
	6. After executing the test and closing the chrome browser
	   Goto ->DemoKatalon/ExtentReports open index.htm in chrome to view the extent report
